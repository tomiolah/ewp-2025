# EWP 2025

This repository contains the music data and related resources for the EWP 2025 worship program, as well as the projection files for use with ProPresenter.

## Sheet music PDF
The sheet music for EWP 2025 is not available as a PDF file in the repository, as it would be too big as file, so you can find it on this [Google Drive link](https://drive.google.com/file/d/1eVWMmHuQwYYf-rzZyTRGNHkF5eK5r0g5/view?usp=sharing).

## YouTube Playlist

A YouTube playlist with all the songs from EWP 2025 can be found [here](https://youtube.com/playlist?list=PLCP4gH5CGC1HNJ7M_StO44WblUZIcnfrW&si=IbISqLog5Dpbb21o).
